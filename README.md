# Split-it!
An expense splitting CLI program.

## Getting Started
1. Make sure you have Node installed
  - You can download it [here](https://nodejs.org/en/).
2. Install dependencies
```
  $ npm i
```
3. Build the program
  - This has to be run every time a change is made
```
  $ npm run tsc
```
4. Start the program
```
  $ npm start
```

## Testing
Run tests:
```
  $ npm test
```
